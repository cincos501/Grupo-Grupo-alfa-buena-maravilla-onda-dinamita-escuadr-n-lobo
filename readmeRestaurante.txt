Instrucciones para correr el proyecto.

Backend:
- descargar el proyecto zip, decomprimirlo

- Para el backend, borrar la carpeta migraciones.
- desde la consola de paquete nugget introducir comandos:
- Add-Migration Initial
- Update - Database

BaseDatos:

- una vez creada la base de datos mediante la migración usar el archivo de inserts con datos para las tablas

Frontend:
- Abrir en vsCode, instalar librerías que requiere al correr con npm run dev
